# User Story: Clear Active Project Filter

**Story ID:** US-004.04
**Epic:** EPIC-004 — Project Focus and Board Maintenance Controls
**Author:** GitHub Copilot
**Status:** Backlog
**Sprint:** Unassigned
**Last Updated:** 2026-05-05

---

## 1. User Story

> As a Maya,
> I want to clear the active project filter in one action,
> so that I can quickly return to a full-board view.

---

## 2. Acceptance Criteria

- [ ] **AC-1:** Given a project filter is active, when I choose Clear Filter, then all tasks across all tags become visible.
- [ ] **AC-2:** Given no filter is active, when I view filter controls, then Clear Filter is disabled or hidden.
- [ ] **AC-3:** Given I clear a filter, when I switch between columns, then no implicit filter is reapplied.

---

## 3. Technical Notes

- **Affected components:** ProjectFilterBar.tsx [Assumed], TaskBoard.tsx [Assumed]
- **API / Data considerations:** Reset selectedTag state to null and re-run board selector.
- **Edge cases to watch:** Clearing filter immediately after selecting a new tag, empty-board behavior after clear.
- **Security / compliance notes:** None.
- **External dependencies:** None.
